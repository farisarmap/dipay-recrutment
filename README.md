# **dipay-recrutment**

## Guidance

1. `npm install`
2. Create .env file with env:
    ```
    - PORT=3000
    - USER=postgres
    - HOST=localhost
    - DB=dipay
    - PASSWORD=postgres
    ```
3. `node .\src\config\db.js`
4. `node .\src\migrations\user.js`
5. `node .\src\migrations\note.js`
6. `npm run dev`
